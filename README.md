# Hunter-Info

A clean, minimal landing page serving as an instant social hub. It introduces who I am with a quick one-line description and provides direct links to my core profiles.

**🌐 Live Site:** [hunter-info.pages.dev](https://hunter-info.pages.dev/)

---

## ✨ Features

- **Minimalist Aesthetic:** A distraction-free, one-page layout built for instant load times.
- **Unified Links:** Quick access to my GitHub, Discord Server, and Instagram in one centralized location.
- **Vanilla Build:** 100% lightweight client-side code with zero external framework dependencies.
